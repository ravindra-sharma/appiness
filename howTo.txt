1. run "node app.js"
2. register new user
3. to see the registered user call:-
	http://localhost:8000/users
   in your browser.
4. to delete all user click reset button.this will help 
you to check that every time on first user registration 
role is "admin".
5. to see all available roles:-
 "http://localhost:8000/roles"
in your browser.
6. 1 - admin
   2 - user
   3 - subuser
   4 - guest
